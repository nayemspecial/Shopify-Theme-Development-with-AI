### ১. ভূমিকা: কেন আমরা এআই-নেটিভ এনভায়রনমেন্ট ব্যবহার করছি?

২০২৬ সালের রিয়েল-ওয়ার্ল্ড থিম ডেভেলপমেন্টে শুধু কোড করতে জানা যথেষ্ট নয়, বরং এআই টুলকে বুদ্ধিমত্তার সাথে কোডিং পার্টনার হিসেবে ব্যবহার করে প্রোডাক্টিভিটি ১০ গুণ বাড়ানোই আমাদের মূল লক্ষ্য। এই মডিউলে আমরা আমাদের ডেভেলপমেন্ট ল্যাব গোছাবো, যেখানে উইন্ডসার্ফ (Windsurf) বা কার্সার (Cursor)-এর মতো এআই-নেটিভ এডিটর সরাসরি আমাদের শপিফাই ও গিটহাব এনভায়রনমেন্টের সাথে কানেক্টেড থাকবে।

### ২. উইন্ডসার্ফ (Windsurf) / এআই কোড এডিটর কনফিগারেশন

ভিডিওতে স্ক্রিন শেয়ার করে এডিটর খোলার পর শিক্ষার্থীদের ধাপে ধাপে সেটিংসগুলো দেখান:

1. **উইন্ডসার্ফ ইন্টিগ্রেশন:** উইন্ডসার্ফ হলো একটি এআই-নেটিভ কোড এডিটর। এর **"Cascade"** বা এআই অ্যাসিস্ট্যান্টকে আমাদের প্রজেক্টের রুট ডিরেক্টরি (Root Directory) রিড করার পারমিশন দেবো।
2. **শপিফাই লিকুইড (Liquid) এক্সটেনশন ইনস্টল:**
* এডিটরের এক্সটেনশন মার্কেটে (`Ctrl+Shift+X` বা `Cmd+Shift+X`) যান।
* **"Shopify Liquid"** লিখে সার্চ করুন এবং অফিশিয়াল এক্সটেনশনটি ইনস্টল করুন। এটি আমাদের লিকুইড সিনট্যাক্স হাইলাইটিং এবং অটো-কমপ্লিশনে সাহায্য করবে।


3. **এআই প্রম্পট রুলস সেটআপ (`.windsurfrules` বা `.cursorrules`):**
* প্রজেক্ট ফোল্ডারে একটি নতুন ফাইল তৈরি করুন যার নাম দিন `.windsurfrules`।
* এই ফাইলে এআইকে ইনস্ট্রাকশন দিন:
> "আমরা শপিফাই অনলাইন স্টোর ২.০ (OS 2.0) স্ট্যান্ডার্ডে কাজ করছি। সব টেমপ্লেট অবশ্যই জেসন (`.json`) আর্কিটেকচারে হবে, কোনো পুরনো `.liquid` টেমপ্লেট ব্যবহার করা যাবে না। সিএসএস ও জেএস-এর জন্য নেটিভ ওয়েব কম্পোনেন্ট এবং মডার্ন স্ট্যান্ডার্ড ফলো করো।"

### ৩. শপিফাই CLI ইন্সটলেশন এবং টার্মিনাল সেটআপ

শপিফাই থিম লোকাল কম্পিউটারে রান ও টেস্ট করার জন্য **Shopify CLI** (Command Line Interface) ইনস্টল করা বাধ্যতামূলক। টার্মিনাল ওপেন করে নিচের কমান্ডগুলো প্র্যাক্টিকালি দেখান:

1. **Node.js এবং Package Manager চেক:** প্রিপারেশন.
শপিফাই CLI রান করার জন্য কম্পিউটারে **Node.js (Version 18 বা তার বেশি)** ইনস্টল থাকতে হবে। টার্মিনালে `node -v` এবং `npm -v` লিখে নিশ্চিত হয়ে নিন।


2. **Shopify CLI ইনস্টল করা:** টার্মিনাল কমান্ড.
আপনার টার্মিনাল (বা উইন্ডসার্ফের ইন-বিল্ট টার্মিনাল) ওপেন করে বিশ্বব্যাপী শপিফাই সিএলআই ইনস্টল করার জন্য নিচের কমান্ডটি রান করুন:

```bash
npm install -g @shopify/cli @shopify/theme

```

*(ইনস্টল শেষ হলে `shopify version` লিখে চেক করুন যে সিএলআই সঠিকভাবে কাজ করছে কিনা।)*


3. **শপিফাই পার্টনার অ্যাকাউন্টে লগইন:** অথেনটিকেশন.
এবার আমাদের লোকাল টার্মিনালকে শপিফাই পার্টনার অ্যাকাউন্টের সাথে কানেক্ট করতে হবে। টার্মিনালে লিখুন:

```bash
shopify login

```

এটি সরাসরি আপনার ব্রাউজার ওপেন করবে। সেখানে আপনার ডেভেলপার মেইল (**nayemku@gmail.com**) দিয়ে লগইন সম্পূর্ণ করুন।

### ৪. এআই টুলের মাধ্যমে গিটহাব (GitHub) ও ভার্সন কন্ট্রোল ম্যানেজমেন্ট

রিয়েল-ওয়ার্ল্ড প্রজেক্টে কোড ব্যাকআপ রাখা এবং শপিফাইয়ের সাথে অটো-সিঙ্ক (Auto-sync) করার জন্য গিটহাব ব্যবহার করা হয়। উইন্ডসার্ফের এআই (Cascade)-কে ব্যবহার করে আমরা এই কাজটি খুব সহজে করবো:

1. **গিটহাব রিপোজিটরি তৈরি:**
* আপনার গিটহাব অ্যাকাউন্টে গিয়ে `projuktiplus-os2-theme` নামে একটি প্রাইভেট বা পাবলিক রিপোজিটরি তৈরি করুন।


2. **এআই-এর সাহায্যে গিট ইনিশিয়ালাইজ করা:**
* উইন্ডসার্ফের Cascade-কে প্রম্পট দিন: *"আমার এই প্রজেক্টের জন্য একটি `.gitignore` ফাইল তৈরি করো যাতে শপিফাইয়ের অপ্রয়োজনীয় কনফিগ ফাইল গিটহাবে না যায়, এবং গিট ইনিশিয়াল করো।"*
* এআই নিজে থেকেই `.gitignore` ফাইল বানিয়ে টার্মিনালে নিচের কমান্ডগুলো রান করে দেবে:
```bash
git init
git add .
git commit -m "Initial commit - AI Native Environment Setup"

```





```
3. **রিমোট রিপোজিটরি পুশ:**
   * গিটহাবের রিমোট লিংকটি যুক্ত করে কোড পুশ করুন:
     ```bash

### ৫. জুন ২০২৬ ওয়ার্কফ্লো: শপিফাই স্টোরের সাথে গিটহাব লিংক করা

OS 2.0-এর সবচেয়ে ম্যাজিকাল পার্ট হলো গিটহাব ইন্টিগ্রেশন।

* আপনার শপিফাই ডেভেলপমেন্ট স্টোরের এডমিন প্যানেলে যান (`[projuktiplus-theme-test.myshopify.com/admin](https://projuktiplus-theme-test.myshopify.com/admin)`)।
* **Online Store -> Themes** সেকশনে যান।
* **Add Theme** বাটনে ক্লিক করে **"Connect from GitHub"** অপশনটি সিলেক্ট করুন।
* আপনার গিটহাব অ্যাকাউন্ট অথোরাইজ করে একটু আগে তৈরি করা `projuktiplus-os2-theme` রিপোজিটরিটির `main` ব্রাঞ্চ কানেক্ট করে দিন।
